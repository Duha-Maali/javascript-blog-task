# Responsive Blog Layout

A responsive blog page built with HTML and CSS as part of a frontend internship training task.

## Concepts Practiced

- Semantic HTML
- CSS variables
- Flexbox
- CSS Grid
- Responsive breakpoints
- Mobile navigation styling
- Font Awesome icons
- Normalize.css

## How to Run

1. Clone or download the repository.
2. Open `index.html` in a browser.
